  • Complete Step for GBMs Regression (Cross Equatorial Northerly Surge - CENS Dataset)
    This summary outlines a comprehensive step-by-step process for implementing Gradient Boosted Machines (GBMs) including XGBoost, LightGBM and CatBoost for Regression on the Cross Equatorial Northerly Surge (CENS) dataset. In this project, I utilized feature selection techniques from SHAP and Catboost, complemented by fine-tuning using the Optuna module. The objective is to predict and analyze the occurrence of CENS phenomena, offering valuable insights for meteorologists and researchers studying tropical weather patterns.
  • Quick Generate Atmospheric Datasets (ERA5)
    Focused on efficiency, this summary outlines a streamlined process for rapidly generating atmospheric datasets using the ERA5 dataset. By providing quick access to atmospheric data, it facilitates research and analysis in meteorology, climate science, and related fields.
  • GBMs Classification for CENS Phenomenon
    Building on the CENS dataset, this summary employs Gradient Boosted Machines (GBMs) for classification tasks related to the Cross Equatorial Northerly Surge (CENS) phenomenon. The classification model aims to accurately categorize instances of CENS, contributing to a better understanding of this meteorological phenomenon.
  • Comparing CONV1D, LSTM, and Hybrid Model for Rainfall Forecasting
    This project involves a comparative analysis of Convolutional Neural Networks (CONV1D), Long Short-Term Memory (LSTM) networks, and CONV1D-LSTM for rainfall forecasting. By evaluating the performance of these models, the project aims to identify the most effective approach for accurate and reliable rainfall predictions, crucial for various applications in meteorology and water resource management.
